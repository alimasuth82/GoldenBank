How to run the code on Visual Studio:


1. Create a folder somewhere in your computer and drag and drop all files from the zip file into that folder.

2. Open Visual Studio Code and click on the first button underneath the VS logo on the top corner right (or press Ctrl-Shift-E or Shift-Command-E on mac).

3. Next, click on "Open Folder" then locate and select the folder where all the files are placed.

4. Run the code.

Note: The program will still run if you just open all Java files in VS Code, but custom logos will not be shown.
